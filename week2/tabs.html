Overview

Website development is simply the creation, build and maintenance of a website. This includes markup (the language developers use to create websites, such as HTML), coding (the programming language), network configuration and CMS (content management system).
Many use "web development" to describe the actual construction of websites. It includes what you find "under the hood," or on the back end of a website, as well as any coding needed for the visible part of your website, or the "front end."
