**Activity Title:** Link and Button Styling with CSS **Objective:** Teach students how to create and style links and buttons using HTML and CSS. **Materials:** - A code editor (e.g., Visual Studio Code, Sublime Text, Notepad++) - Web browser (for testing) - Internet connection (optional, for online resources) **Instructions:** **Step 1: Setup** Before starting the activity, make sure everyone has access to a code editor and a web browser. Ensure that students understand the basics of HTML and CSS. **Step 2: HTML Structure** Begin by creating an HTML file. Instruct students to create a basic HTML structure:

Example Coding Box


    // This is a sample code snippet

  • Home

  • Contact
``` **Step 4: Basic Button Styling** Teach students how to create a basic button using an HTML ` ``` **Step 5: Hover Effect** Introduce students to the concept of hover effects. Modify the CSS to include a hover effect for both links and buttons: ```css /* Hover effect for links */ a:hover { color: #e74c3c; text-decoration: underline; } /* Hover effect for buttons */ button:hover { background-color: #e74c3c; } ``` **Step 6: Additional Styling** Encourage students to explore more styling options. They can modify fonts, add borders, change text alignment, etc., to further enhance the appearance of links and buttons. **Step 7: Testing** Instruct students to open their HTML file in a web browser to see the results. They can click on the links and interact with the buttons to ensure everything is working as expected. **Step 8: Challenges (Optional)** For more advanced students or as an extension to the activity, you can provide challenges like: - Create a dropdown menu in the navigation. - Style the links to look like buttons. - Add transition effects to the hover states. - Create a responsive design for mobile devices. **Step 9: Share and Discuss** Ask students to share their work with the class and discuss the different styles and effects they applied. Encourage questions and troubleshooting if anyone encounters issues. This activity will help students gain hands-on experience with HTML and CSS while focusing on links and buttons, which are fundamental elements of web development.
