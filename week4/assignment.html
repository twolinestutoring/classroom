Accordion FAQ

Objective:

An "Accordion FAQ" project involves creating a webpage with a frequently asked questions (FAQ) section where users can click on a question to reveal its answer. When one question is expanded, the others should collapse, giving it the accordion effect. Below is a step-by-step guide on how to create this project:

Instructions:

Step 1: HTML

The HTML structure includes a list of questions and answers. Initially, all answer sections are hidden (set to display: none in the CSS).


            
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Accordion FAQ</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="faq">
        <div class="question">
            <h2>Question 1</h2>
        </div>
        <div class="answer">
            <p>Answer to Question 1 goes here.</p>
        </div>
        
        <div class="question">
            <h2>Question 2</h2>
        </div>
        <div class="answer">
            <p>Answer to Question 2 goes here.</p>
        </div>
        
        <!-- Add more questions and answers as needed -->
    </div>
    <script src="script.js"></script>
</body>
</html>

Step 2: CSS

CSS is used to style the questions as clickable buttons. When a question is clicked, it becomes "active" and is highlighted. You can customize the styling to match your design preferences.


/* Hide all answer sections by default */
.answer {
    display: none;
}

/* Style for questions to make them look like clickable buttons */
.question {
    background-color: #f0f0f0;
    padding: 10px;
    cursor: pointer;
}

/* Style for active questions */
.question.active {
    background-color: #ccc;
}

Step 3: Add JavaScript

JavaScript is used to add interactivity. It listens for click events on the questions. When a question is clicked, it toggles the 'active' class to change the styling, and it toggles the display of the associated answer section. This creates the accordion effect.


const questions = document.querySelectorAll('.question');

questions.forEach((question) => {
    question.addEventListener('click', () => {
        // Toggle the 'active' class for the clicked question
        question.classList.toggle('active');

        // Get the associated answer element
        const answer = question.nextElementElement;

        // Toggle the display of the answer
        if (question.classList.contains('active')) {
            answer.style.display = 'block';
        } else {
            answer.style.display = 'none';
        }
    });
});
