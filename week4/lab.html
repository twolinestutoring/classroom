We will be creating an accordian using FAQ's about Computer Engeneering.

    What is computer engineering?
    Computer engineering is a discipline that combines electrical engineering and computer science. It focuses on designing and developing computer systems, hardware, and software.

    What's the difference between computer engineering and computer science?
    Computer engineering primarily deals with designing and building computer hardware and embedded systems, whereas computer science focuses on software development and algorithms.

    What are the career opportunities in computer engineering?
    Computer engineers can work in various fields, including hardware design, software development, network engineering, robotics, embedded systems, and cybersecurity.

    What skills are essential for a computer engineering career?
    Essential skills include programming languages (e.g., C, C++, Java), knowledge of digital systems, circuit design, problem-solving, and understanding of algorithms and data structures.

    How do computer engineers contribute to the tech industry?
    Computer engineers play a crucial role in developing the latest technologies, including smartphones, computer processors, IoT devices, and advanced software applications.

    What's the future of computer engineering?
    The computer engineering field continues to evolve, with growth in areas like artificial intelligence, cybersecurity, and automation, offering promising career opportunities.

    What are some emerging trends in computer engineering?
    Emerging trends include edge computing, quantum computing, 5G technology, Internet of Things (IoT), and machine learning.

    Do computer engineers need to stay updated with the latest technologies?
    Yes, staying current with advancements in technology is essential for computer engineers, as the industry continually evolves.

    How do I become a computer engineer?
    To become a computer engineer, you typically need a bachelor's degree in computer engineering, electrical engineering, or a related field. Further education or certifications can enhance career prospects.

    What are some notable challenges in computer engineering?
    Challenges include addressing security threats, keeping up with rapid technological changes, and designing energy-efficient hardware.

    Are internships important for computer engineering students?
    Yes, internships provide practical experience and networking opportunities. They can significantly enhance a student's chances of securing a job after graduation.

    What's the role of computer engineers in sustainability and environmental concerns?
    Computer engineers work on developing energy-efficient hardware and software, contributing to reducing the environmental impact of technology.

Step 2: Make you CSS

you can change the look of your accordian using the CSS. Use the folowing code is a guide, but make changes to the colors. You can also add borders or box shadow to experiment more with CSS.

CSS is used to style the questions as clickable buttons. When a question is clicked, it becomes "active" and is highlighted. You can customize the styling to match your design preferences.


/* Hide all answer sections by default */
.answer {
    display: none;
}

/* Style for questions to make them look like clickable buttons */
.question {
    background-color: #f0f0f0;
    padding: 10px;
    cursor: pointer;
}

/* Style for active questions */
.question.active {
    background-color: #ccc;
}

Step 3: Add JavaScript

JavaScript is used to add interactivity. It listens for click events on the questions. When a question is clicked, it toggles the 'active' class to change the styling, and it toggles the display of the associated answer section. This creates the accordion effect.

each of the folowing lines introduces new variables and js elements. Make sure to watch this weeks video to better understand js.


const questions = document.querySelectorAll('.question');

questions.forEach((question) => {
    question.addEventListener('click', () => {
        // Toggle the 'active' class for the clicked question
        question.classList.toggle('active');

        // Get the associated answer element
        const answer = question.nextElementElement;

        // Toggle the display of the answer
        if (question.classList.contains('active')) {
            answer.style.display = 'block';
        } else {
            answer.style.display = 'none';
        }
    });
});
